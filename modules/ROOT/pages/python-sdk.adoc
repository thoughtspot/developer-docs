ThoughtSpot provides a Python SDK for REST API v2.0, enabling Python developers to integrate ThoughtSpot data operations and analytics workflows in their applications without writing raw HTTP requests. The SDK wraps the ThoughtSpot REST API v2.0 endpoints into a typed Python client, reducing boilerplate and simplifying common operations such as authentication, Liveboard export, data search, and user management.

[NOTE]
====
The Python SDK for REST API v2.0 is available starting from the ThoughtSpot Cloud jul.26.mt release. For the latest REST API information, see xref:rest-apiv2-changelog.adoc[REST API v2.0 changelog].
====
